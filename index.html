Technical Roles

The engineers, designers, and analysts who build and protect our systems.

Software Developer / Software Engineer
Build software applications, websites, and systems.
Responsibilities: Write and test code, develop web and mobile apps, fix bugs, integrate databases & APIs, maintain existing software.
Skills: Java, Python, JavaScript, PHP, C#, problem-solving, database management.
JavaPythonJS
Front-End Developer
Create the part of a website or app users interact with.
Responsibilities: Design user interfaces, build responsive pages, improve UX, work with designers and backend developers.
Tools: HTML, CSS, JavaScript, React, Angular.
HTMLCSSReact
Back-End Developer
Build and manage the server-side logic of applications.
Responsibilities: Develop APIs, manage databases, handle authentication & security, ensure system performance.
Tools: Node.js, Java, Python, PostgreSQL, MySQL.
Node.jsPythonSQL
Full-Stack Developer
Handle both front-end and back-end development.
Responsibilities: Build complete applications, connect interfaces with databases, deploy and maintain systems, troubleshoot issues.
Note: Kenyan startups prefer full-stack developers because they can work across the entire product.
Full StackDevOps
Quality Assurance (QA) Analyst / Software Tester
Ensure software works correctly before release.
Responsibilities: Create test plans, perform manual & automated testing, report bugs, verify fixes, improve software quality.
TestingSeleniumCypress

Responsibilities: Monitor security threats, conduct vulnerability assessments, perform penetration testing, implement security controls, investigate incidents.

The strategists, coordinators, and client-facing professionals who drive growth.

Project Manager (PM)
Ensure projects are delivered on time and within budget.
Responsibilities: Plan project schedules, coordinate teams, monitor progress, manage risks & budgets, communicate with clients and stakeholders.
Business Analyst (BA)
Connect business needs with technical solutions.
Responsibilities: Gather client requirements, analyze business processes, create project documentation, support testing & implementation, recommend improvements.
Technical Support / IT Support Officer

Responsibilities: Install software, resolve computer issues, assist employees and customers, maintain IT equipment.
